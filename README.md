# Velvet & Vine Restaurant

Velvet & Vine is a warm, Nepali-inspired restaurant experience brought to life on the web. This Vite + React application showcases the restaurant's signature dishes, curated thali offerings, and welcoming ambiance while giving visitors quick access to reservations, location details, and WhatsApp ordering.
